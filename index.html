Preguntas Frecuentes

¿Tienes Dudas?

¿Necesito experiencia previa con IA? +
No. NeuroClick está diseñado para personas sin experiencia técnica. La Clase 0 es precisamente para determinar tu punto de partida real y construir desde ahí.
¿Cuál es la diferencia entre síncrono y asincrónico? +
El síncrono son encuentros en vivo 1:1 por Zoom conmigo — en tiempo real, con preguntas y práctica guiada. El asincrónico son clases grabadas que puedes ver cuando quieras durante 3 meses, con 2 asesorías incluidas para resolver dudas.
¿Qué pasa si elijo síncrono y no puedo en ese horario? +
Coordinas directamente con Ada. Si tienes un imprevisto, reagendamos. El programa 1:1 tiene esa flexibilidad.
¿Puedo cambiar de nivel después? +
Sí. Puedes avanzar a VideoClick™, SoundClick™, VisualClick™ o Masterclick cuando termines tu nivel actual. Cada nivel incluye una Clase 0 de actualización como bono de bienvenida.
¿En qué moneda es el pago y qué métodos aceptan? +
El precio es $150 USD. Aceptamos PayPal (con o sin cuenta), Wise para transferencias internacionales, y DolarApp para México. También puedes escribir por WhatsApp para coordinar.
